# dkp

`dkp` is the command-line tool for managing Domain Knowledge Packs — structured bundles of curated domain knowledge for AI agents, LLM applications, and human readers.

## What You Can Do

- **Author** packs with `dkp init`, `dkp new`, `dkp generate`
- **Validate** conformance with `dkp validate` (8-gate quality standard)
- **Search and retrieve** content with `dkp search`, `dkp chunk`, `dkp inject`
- **Export** to OKF, LangChain, LlamaIndex, and other formats with `dkp export`
- **Evaluate** pack quality against a grounded eval set with `dkp eval`
- **Publish** and distribute packs via `dkp publish`, `dkp install`
- **Serve** a pack as an MCP server with `dkp serve`

See the [Command Reference](reference/cli-reference.md) for the full list of commands and flags.
